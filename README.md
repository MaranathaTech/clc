# Compliment Me

A beautiful Next.js app that gives you a sweet personalized compliment when you submit a simple form.

## Features

- Elegant gradient UI with glass-morphism card design
- Optional name input for personalized compliments
- 20 unique compliments with creative personalization
- Loading state with smooth feedback
- Dark mode support
- Fully responsive design

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and get complimented!

## Tech Stack

- **Next.js 16** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS 4** - Styling
